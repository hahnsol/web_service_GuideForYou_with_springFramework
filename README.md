## 1. 프로젝트 명
> Guid For You  

<br>

## 2. 프로젝트 목적
> 다양한 테마투어 선택 및 비슷한 성향의 사람들과 커뮤니티를 형성할 수 있는 공간 창출 목표

<br>

## 3. 프로젝트 환경
<pre>
<code>
* WEB 
1. OS : ubuntu 
2. Spring Framework
3. 언어: Java

* DB
1. Oracle DB

* 배포
1. AWS EC2
</code>
</pre>

<br>

## 4. 담당 역할
> 프로젝트 기획, DB 설계, index, main, 투어 등록 및 예약, 관리 등의 back-end 담당과 전체 페이지 front-end 담당
